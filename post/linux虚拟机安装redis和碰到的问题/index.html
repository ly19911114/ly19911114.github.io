Linux虚拟机安装Redis和碰到的问题

Posted on Feb 20, 2020

1、解决could not retrieve mirrorlist，

          1.sudo 
          2.sudo su用sudo命令进入管理员账号
          vim /etc/sysconfig/network-scripts/ifcfg-ens33 (不一定是ens33，要看清楚自己的文件名字)

　　 3.将ONBOOT改为yes，wq!保存退出

          4.重新启动网络  $ service network restart

​
2、 安装wget yum install wget

​
3、 下载redis安装包

         wget http://download.redis.io/releases/redis-4.0.6.tar.gz

​
4、 解压压缩包

         tar zxvf redis-4.0.6.tar.gz

​
5、yum install gcc

​
6、跳转到redis解压目录下 cd redis-4.0.6

​
7、编译安装 make MALLOC=libc　　

     8、cd src && make install

     9、测试是否安装成功 

          ./redis-server
